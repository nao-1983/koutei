# 現場工程表

現場名称・工期・工事項目を入力すると、色付きラインの工程表(ガントチャート)を表示するWebアプリです。
ビルド不要。ファイルを置くだけで動きます。

## ファイル

- `index.html` … アプリ本体(これだけでも動きます)
- `sw.js` … 圏外でも開けるようにするためのキャッシュ用ファイル

## 機能

- 現場名称、着工〜竣工の入力
- 工事項目の追加・編集・削除(名称・開始日・終了日・色・備考)
- JIS安全色ベースの6色でラインを色分け
- 土日の色分け、本日位置に赤線
- 備考欄の表示/非表示
- PDF書き出し(A4横)
- 入力内容はブラウザに自動保存(localStorage)
- データの書き出し/読み込み(JSON)
- iPad向け:入力欄のズーム防止、指で押せるサイズ、オフライン起動

## iPadだけで公開する手順(PC不要)

1. Safariで github.com を開き、ログイン(アカウントがなければ作成)
2. `+` → `New repository` → 名前を `koutei` にして **Public** で作成
3. `uploading an existing file` をタップ → `index.html` と `sw.js` と `README.md` を選んでアップロード → `Commit changes`
   - ファイルは事前に「ファイル」アプリに保存しておいてください
4. `Settings` → `Pages` → Source を `Deploy from a branch`、Branch を `main` / `/ (root)` にして `Save`
5. 1〜2分待って `https://<ユーザー名>.github.io/koutei/` をSafariで開く
6. 共有ボタン → **ホーム画面に追加** → アプリのように全画面で起動できます

## 注意

- データは**その端末のブラウザ内**に保存されます。iPadとPCで共有したい場合は「データを書き出す」でJSONを保存し、別端末で「データを読み込む」を使ってください
- ホーム画面に追加したアプリと、Safariのタブで開いた場合とで、保存データは共有されます(同じサイトのため)
- Safariの「履歴とWebサイトデータを消去」を実行すると保存データも消えます。大事な工程表はJSONで書き出しておくと安全です
